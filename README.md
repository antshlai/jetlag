# Jetlag: The Game - Digital
This repository hosts original board files in CSV format for both Hide and Seek and Captured game versions. 

The public facing game interface is accessible at [jetlag.oneverse.au](https://jetlag.oneverse.au). 

If you would like to self-host your own Jetlag game console, please get in touch and the source code can be made available to you.

This project takes inspiration from Jetlag: The Game YouTube series, but is in no way associated with the creators. It is not endorsed, produced or provided by the original creators. The board cards remain the intellectual property of the creators.

The self-hosted software of the game is Copyright and thus, not hosted on GitHub. An open source version may be made available later, however, there is a general version available at [this link](https://www.lifack.ch).
